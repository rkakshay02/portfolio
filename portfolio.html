Movie Recommendation System

A content-based movie recommendation system that uses cosine similarity to suggest films based on a user's preferences. The system analyzes movie features like genre, cast, director, and keywords to recommend similar movies.

Technologies used: Python, Pandas, NumPy, Scikit-learn (for cosine similarity calculations), TMDB API for movie data, and Streamlit for a simple user interface.
